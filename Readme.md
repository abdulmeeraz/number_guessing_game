## Description
A Python command-line number guessing game where the player must guess a randomly generated number within a limited number of attempts.  
The game automatically replays based on the user's choice.

## Features
- Random number generation
- Limited number of attempts
- Hints for high/low guesses
- Replay option (Y/N)

## Concepts Used
- Python functions
- Loops
- Conditional statements
- Random module
